# AI Multi-Asset Investment Copilot

An AI-native investment research assistant powered by Claude, designed to provide multi-dimensional analysis for mid-to-long-term investment strategies. This system automates the ingestion and analysis of macroeconomic data, fundamental reports, market sentiment, and technical indicators across **Equities, Indices, and Crypto**.

Key Features

*   **Automated Research Pipeline**: Triggered by natural language prompts, the system automatically scrapes market data and generates professional-grade analysis reports.
*   **Multi-Dimensional Analysis**:
    *   **Macro & Fundamentals**: Parses top-tier investment bank research and macroeconomic indicators.
    *   **Market Sentiment**: Analyzes social sentiment and news flows.
    *   **Technical Analysis**: Executes statistical modeling and technical indicator analysis.
*   **Asset Coverage**: Provides deep-dive insights into Global Indices, Blue-chip Stocks, and Cryptocurrency assets.
*   **AI-Native Workflow**: Leverages advanced LLMs (Claude) to reduce manual research time by 90% while maintaining high analytical rigour.

## 🛠 Tech Stack

*   **Core Engine**: Claude (LLM) for reasoning and code generation.
*   **Data Acquisition**: Python-based automated web scraping and API integration.
*   **Environment**: Terminal-based command execution and cloud-ready architecture.

## 📊 How it Works

1.  **Input**: The user provides a high-level investment idea or a specific ticker via a terminal command.
2.  **Orchestration**: The system generates and executes Python scripts to fetch real-time data from multiple sources.
3.  **Analysis**: The LLM synthesizes the data, applying investment logic to generate a structured "Investment Thesis" report.
4.  **Output**: A concise, actionable report containing actionable insights and risk assessments.
